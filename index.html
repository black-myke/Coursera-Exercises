Css Positioning

Static

Static Positioning is the default position for HTML elements.
Elements with the position: Static are positioned based on the normal flow of the page, as you would expecct them to be without any CSS.

Relative

Relative Positioning is a postion property that the element is positioned relative to its normal position. The offsets (top, bottom, left, right) will cause the element to be adjusted from its normal position but would not be taken from its normal flow.

Absolute

Absolute Positioning is a postion property that takes the targeted elements out of its normal document flow.
The offsets (top, bottom, left, right) are relative to the position of the nearest ancestor which has a positioning set on it.
